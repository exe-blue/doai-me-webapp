## 3. Product: DoAi.Me MVP

### 핵심 기능

#### 1차 콘텐츠 소비 (YouTube 시청)
- YouTube URL 입력으로 작업 생성
- 시청 시간 범위 설정 (30%~90% 랜덤)
- 실시간 진행률 모니터링

#### 불확실성 엔진
- 좋아요/댓글/저장 확률 설정
- 각 디바이스가 독립적으로 결정
- 자연스러운 행동 패턴 생성

#### 에러 복구 시스템
- 체계적인 에러 코드 (E1xxx~E4xxx)
- Exponential Backoff 재시도 전략
- 앱 크래시 자동 복구

### 기술 스택

```
Central Server: Next.js 16 + Supabase + Tailscale VPN
Mobile Agent: AutoX.js + REST API Client
Hardware: Galaxy S9 x 600 + LTE SIM x 600
```
